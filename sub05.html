Combine Find Movie

영화 시청 및 스크랩 앱

Duration: 2025.05 (2일)
Role: UI/UX Design 100%
Tool: Figma, Photoshop
Responsive: Desktop / Tablet / Mobile

Project Overview

짝꿍의 아이디어를 디자인으로 구현한 협업 프로젝트입니다. 좋아하는 배우와 작품을 저장하면 취향 기반으로 영화를 추천해주는 앱으로, OCN 모바일 앱을 레퍼런스로 참고해 UI 패턴을 분석했습니다. 팀원이 홀로그램 톤을 원했지만 영화 포스터가 주인공인 앱에서 배경이 튀면 콘텐츠 집중도가 떨어진다고 설득해 다크톤으로 결정했습니다.

Main Screen

박스오피스 순위, 공개 예정작, 추천 영화를 한 화면에서 탐색할 수 있도록 카드 레이아웃으로 구성했습니다. 취향 맞춤 앱의 특성을 고려해 "오늘 영화 뭐 보지?" 섹션을 별도로 배치해 추천 콘텐츠가 자연스럽게 노출되도록 했습니다.

Search Screen

빈 검색창 상태에서도 최근 검색 기록과 인기 검색어를 노출해 탐색이 자연스럽게 이어지도록 구성했습니다.

Favorite Screen

좋아하는 배우, 좋아하는 작품, 관심 작품, AI 추천 작품으로 분류해 스크랩한 콘텐츠를 취향별로 탐색할 수 있도록 구성했습니다.
